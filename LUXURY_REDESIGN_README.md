# 🏨 表参道サロン - 高級ペットサロン管理システム

## 📋 プロジェクト概要

既存のペットサロン受付アプリを**高級ホテルスパ風**のUI/UXに完全リニューアルしました。「清潔・安心・落ち着き・上質」をテーマに、管理者・スタッフが毎日の業務で疲れない洗練されたデザインシステムを構築しています。

## 🎨 デザインシステム

### カラーパレット

- **Primary**: 既存の淡いブルー系グラデーションをベースにした上品な配色
- **Secondary**: 落ち着いたパープル系のアクセント
- **Neutral**: 高級感のあるグレー階調
- **Status**: 意味の分かりやすい警告色・成功色

### タイポグラフィ

- **Primary Font**: Noto Sans JP (日本語最適化)
- **Secondary Font**: Inter (欧文・数字)
- **Font Scale**: Golden Ratio ベースの美しい文字サイズ階層
- **Line Height**: 日本語テキストに最適化された行間

### UI コンポーネント

- **角丸**: 12-16px で統一
- **シャドウ**: ソフトシャドウで高級感演出
- **余白**: 4pt グリッドベースの整然とした配置
- **アニメーション**: 上品で控えめなマイクロインタラクション

- **売上・物販**: 売上・物販・在庫
- **運営・設定**: スタッフ・イベント・店舗設定

## 📊 主要機能

### 1. ダッシュボード

- **今日の予約**: 時系列で表示、担当スタッフ・ステータス管理
- **注意ペット**: 薬・アレルギー・注意事項の視覚的表示
- **売上サマリー**: 期間別の施術・物販売上とグラフ表示
- **カルテ更新**: 最近の更新履歴とクイックアクセス

### 2. 詳細画面設計

- **顧客プロフィール**: 左側固定プロフィール + 右側タブ切り替え
- **予約管理**: カレンダービュー/リストビュー切り替え
- **カルテ編集**: 構造化されたフォームと入力補助
- **売上管理**: グラフ + テーブルの組み合わせ

### 3. レスポンシブ対応

- **PC版**: 280px サイドバー + メインコンテンツの2カラム
- **タブレット**: サイドバー非表示、ヘッダーナビゲーション
- **モバイル**: タッチフレンドリーな操作性、48px以上のタップターゲット
